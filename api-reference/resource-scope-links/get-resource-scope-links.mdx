---
title: 'Get Resource Scope Links'
api: 'GET /resource-scope-links'
description: 'Get resource scope links by resource or scope'
---

## Query Parameters

<ParamField query="resourceId" type="string">
  Filter by resource ID
</ParamField>

<ParamField query="scopeId" type="string">
  Filter by scope ID
</ParamField>

<ParamField query="linkType" type="string">
  Filter by link type: `share`, `alias`, or `mirror`
</ParamField>

<Note>
  At least one of `resourceId` or `scopeId` is required.
</Note>

## Response

Returns an array of resource scope link objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/resource-scope-links?resourceId=resource_doc_123' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
