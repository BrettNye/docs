---
title: 'Delete Role'
api: 'DELETE /roles/{id}'
description: 'Delete a role from the system'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the role to delete
</ParamField>

## Response

Returns a success confirmation.

<Warning>
  Deleting a role will also remove all role assignments and role-permission mappings associated with it.
</Warning>

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/roles/role_123' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```

```javascript Node.js
const axios = require('axios');

const { data } = await axios.delete('https://api.example.com/roles/role_123', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
});
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true
}
```
</ResponseExample>
