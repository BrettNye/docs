---
title: 'Delete Role Permission by Role and Permission'
api: 'DELETE /role-permissions/{roleId}/{permissionId}'
description: 'Remove a permission from a role by specifying both IDs'
---

## Path Parameters

<ParamField path="roleId" type="string" required>
  The role ID
</ParamField>

<ParamField path="permissionId" type="string" required>
  The permission ID to remove from the role
</ParamField>

## Response

Returns a success confirmation.

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/role-permissions/role_123/perm_456' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```

```javascript Node.js
const axios = require('axios');

const { data } = await axios.delete('https://api.example.com/role-permissions/role_123/perm_456', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
});
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true
}
```
</ResponseExample>
