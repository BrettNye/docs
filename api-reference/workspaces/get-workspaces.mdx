---
title: 'Get Workspaces'
api: 'GET /workspaces'
description: 'List workspaces with optional filters'
---

## Query Parameters

<ParamField query="tenantId" type="string">
  Filter by tenant ID
</ParamField>

## Response

Returns an array of workspace objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/workspaces?tenantId=tenant_acme' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
