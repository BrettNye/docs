---
title: 'List Workspaces by Tenant'
api: 'GET /workspaces/tenant/{tenantId}'
description: 'List all workspaces in a tenant'
---

## Path Parameters

<ParamField path="tenantId" type="string" required>
  The tenant ID
</ParamField>

## Response

Returns an array of workspace objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/workspaces/tenant/tenant_acme' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>

<ResponseExample>
