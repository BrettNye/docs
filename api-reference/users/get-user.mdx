---
title: 'Get User'
api: 'GET /users/{id}'
description: 'Get a user by ID'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The user ID
</ParamField>

## Response

Returns the user object.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/users/user_jane' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
