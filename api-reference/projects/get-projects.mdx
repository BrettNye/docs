---
title: 'List Projects by Tenant'
api: 'GET /projects/tenant/{tenantId}'
description: 'List all projects in a tenant'
---

## Path Parameters

<ParamField path="tenantId" type="string" required>
  The tenant ID
</ParamField>

<Note>
  Projects can also be listed by workspace: `GET /projects/workspace/{workspaceId}`.
</Note>

## Response

Returns an array of project objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/projects/tenant/tenant_acme' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
