---
title: 'Get Projects'
api: 'GET /projects'
description: 'List projects with optional filters'
---

## Query Parameters

<ParamField query="tenantId" type="string">
  Filter by tenant ID
</ParamField>

<ParamField query="workspaceId" type="string">
  Filter by workspace ID
</ParamField>

## Response

Returns an array of project objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/projects?workspaceId=workspace_main' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
