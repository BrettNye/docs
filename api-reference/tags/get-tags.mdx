---
title: 'Get Tags'
api: 'GET /tags'
description: 'Get tags for a tag group'
---

## Query Parameters

<ParamField query="tagGroupId" type="string" required>
  ID of the tag group to get tags for
</ParamField>

## Response

Returns an array of tag objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/tags?tagGroupId=tag_group_priority' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
