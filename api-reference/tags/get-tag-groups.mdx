---
title: 'Get Tag Groups'
api: 'GET /tag-groups'
description: 'Get tag groups for a scope'
---

## Query Parameters

<ParamField query="scopeId" type="string" required>
  ID of the scope to get tag groups for
</ParamField>

## Response

Returns an array of tag group objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/tag-groups?scopeId=scope_acme' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
