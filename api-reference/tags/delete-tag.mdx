---
title: 'Delete Tag'
api: 'DELETE /tags/{id}'
description: 'Delete a tag'
---

## Path Parameters

<ParamField path="id" type="string" required>
  ID of the tag to delete
</ParamField>

## Response

Returns no content on success.

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/tags/tag_high' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>

<ResponseExample>
```json Response
{}
```
</ResponseExample>
