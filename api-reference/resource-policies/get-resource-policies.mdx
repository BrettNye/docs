---
title: 'Get Resource Policies'
api: 'GET /resource-policies'
description: 'Get resource policies for a scope'
---

## Query Parameters

<ParamField query="scopeId" type="string" required>
  ID of the scope to get policies for
</ParamField>

<ParamField query="resourceId" type="string">
  Filter by policies targeting a specific resource
</ParamField>

<ParamField query="collectionId" type="string">
  Filter by policies targeting a specific collection
</ParamField>

<ParamField query="effect" type="string">
  Filter by effect: `allow` or `deny`
</ParamField>

## Response

Returns an array of resource policy objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/resource-policies?scopeId=scope_org' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
