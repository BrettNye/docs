---
title: 'Delete Role Assignment'
api: 'DELETE /role-assignments/{id}'
description: 'Delete a role assignment by its ID'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the role assignment to delete
</ParamField>

## Response

Returns a success confirmation.

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/role-assignments/ra_123' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```

```javascript Node.js
const axios = require('axios');

const { data } = await axios.delete('https://api.example.com/role-assignments/ra_123', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
});
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true
}
```
</ResponseExample>
