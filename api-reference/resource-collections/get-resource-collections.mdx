---
title: 'Get Resource Collections'
api: 'GET /resource-collections'
description: 'Get resource collections for a scope'
---

## Query Parameters

<ParamField query="scopeId" type="string" required>
  ID of the scope to get collections for
</ParamField>

<ParamField query="resourceTypeId" type="string">
  Filter by resource type
</ParamField>

## Response

Returns an array of resource collection objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/resource-collections?scopeId=scope_org' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
