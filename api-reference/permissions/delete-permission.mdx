---
title: 'Delete Permission'
api: 'DELETE /permissions/{id}'
description: 'Delete a permission from the system'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the permission to delete
</ParamField>

## Response

Returns a success confirmation.

<Warning>
  Deleting a permission will also remove all role-permission mappings associated with it.
</Warning>

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/permissions/perm_123' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```

```javascript Node.js
const axios = require('axios');

const { data } = await axios.delete('https://api.example.com/permissions/perm_123', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
});
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true
}
```
</ResponseExample>
