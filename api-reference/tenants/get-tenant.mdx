---
title: 'Get Tenant'
api: 'GET /tenants/{id}'
description: 'Get a tenant by ID'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The tenant ID
</ParamField>

## Response

Returns the tenant object.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/tenants/tenant_acme' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
