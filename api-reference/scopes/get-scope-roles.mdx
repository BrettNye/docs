---
title: 'Get Scope Roles'
api: 'GET /scopes/{id}/roles'
description: 'Retrieve all roles defined within a scope'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The scope ID
</ParamField>

## Response

Returns an array of role objects belonging to the scope.

<ResponseField name="roles" type="array">
  <Expandable title="Role object">
    <ResponseField name="id" type="string">
      Unique identifier for the role
    </ResponseField>
    <ResponseField name="name" type="string">
      Display name of the role
    </ResponseField>
    <ResponseField name="description" type="string">
      Description of the role
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/scopes/scope_123/roles' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```

```javascript Node.js
const axios = require('axios');

const { data } = await axios.get('https://api.example.com/scopes/scope_123/roles', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
});
```
</RequestExample>
