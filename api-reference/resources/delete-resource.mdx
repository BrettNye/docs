---
title: 'Delete Resource'
api: 'DELETE /resources/{id}'
description: 'Delete a resource'
---

## Path Parameters

<ParamField path="id" type="string" required>
  ID of the resource to delete
</ParamField>

## Response

Returns no content on success.

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/resources/resource_0190a1b2-c3d4-7e5f-8a9b-0c1d2e3f4a5b' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>

<ResponseExample>
```json Response
{}
```
</ResponseExample>
