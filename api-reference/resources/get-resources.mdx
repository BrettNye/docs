---
title: 'Get Resources'
api: 'GET /resources'
description: 'Get resources for a scope'
---

## Query Parameters

<ParamField query="scopeId" type="string" required>
  ID of the scope to get resources for
</ParamField>

## Response

Returns an array of resource objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/resources?scopeId=scope_engineering' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
