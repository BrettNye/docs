---
title: 'Get Resource'
api: 'GET /resources/{id}'
description: 'Get a resource by ID'
---

## Path Parameters

<ParamField path="id" type="string" required>
  ID of the resource
</ParamField>

## Response

Returns the resource object.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/resources/resource_0190a1b2-c3d4-7e5f-8a9b-0c1d2e3f4a5b' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "id": "resource_0190a1b2-c3d4-7e5f-8a9b-0c1d2e3f4a5b",
  "resourceTypeId": "resource_type_document",
  "scopeId": "scope_engineering",
  "externalResourceId": "doc-123",
  "displayName": "Project Roadmap",
  "createdAt": "2024-01-15T10:30:00Z",
  "createdBy": "user-1"
}
```
</ResponseExample>
