---
title: 'Delete Scope Hierarchy Edge'
api: 'DELETE /scope-hierarchy/{parentScopeId}/{childScopeId}'
description: 'Remove a parent-child relationship between two scopes'
---

## Path Parameters

<ParamField path="parentScopeId" type="string" required>
  The parent scope ID
</ParamField>

<ParamField path="childScopeId" type="string" required>
  The child scope ID
</ParamField>

## Response

Returns a success confirmation.

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/scope-hierarchy/scope_org/scope_team' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```

```javascript Node.js
const axios = require('axios');

const { data } = await axios.delete('https://api.example.com/scope-hierarchy/scope_org/scope_team', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
});
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true
}
```
</ResponseExample>
