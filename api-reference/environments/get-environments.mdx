---
title: 'List Environments by Tenant'
api: 'GET /environments/tenant/{tenantId}'
description: 'List all environments in a tenant'
---

## Path Parameters

<ParamField path="tenantId" type="string" required>
  The tenant ID
</ParamField>

<Note>
  Environments can also be listed by project: `GET /environments/project/{projectId}`.
</Note>

## Response

Returns an array of environment objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/environments/tenant/tenant_acme' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
