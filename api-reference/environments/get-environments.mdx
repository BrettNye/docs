---
title: 'Get Environments'
api: 'GET /environments'
description: 'List environments with optional filters'
---

## Query Parameters

<ParamField query="tenantId" type="string">
  Filter by tenant ID
</ParamField>

<ParamField query="projectId" type="string">
  Filter by project ID
</ParamField>

## Response

Returns an array of environment objects.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/environments?projectId=project_api' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>
