---
title: 'Delete Role Override by Scope and Role'
api: 'DELETE /scope-overrides/roles/{childScopeId}/{roleId}'
description: 'Delete a role override by specifying the scope and role'
---

## Path Parameters

<ParamField path="childScopeId" type="string" required>
  The child scope ID
</ParamField>

<ParamField path="roleId" type="string" required>
  The role ID
</ParamField>

## Response

Returns a success confirmation.

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/scope-overrides/roles/scope_team/role_admin' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```

```javascript Node.js
const axios = require('axios');

const { data } = await axios.delete('https://api.example.com/scope-overrides/roles/scope_team/role_admin', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
});
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true
}
```
</ResponseExample>
