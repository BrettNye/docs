---
title: 'Delete Role-Permission Override by IDs'
api: 'DELETE /scope-overrides/role-permissions/{childScopeId}/{roleId}/{permissionId}'
description: 'Delete a role-permission override by specifying scope, role, and permission'
---

## Path Parameters

<ParamField path="childScopeId" type="string" required>
  The child scope ID
</ParamField>

<ParamField path="roleId" type="string" required>
  The role ID
</ParamField>

<ParamField path="permissionId" type="string" required>
  The permission ID
</ParamField>

## Response

Returns a success confirmation.

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/scope-overrides/role-permissions/scope_team/role_editor/perm_delete' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```

```javascript Node.js
const axios = require('axios');

const { data } = await axios.delete('https://api.example.com/scope-overrides/role-permissions/scope_team/role_editor/perm_delete', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
});
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true
}
```
</ResponseExample>
