---
title: 'Delete Permission Override'
api: 'DELETE /scope-overrides/permissions/{id}'
description: 'Delete a permission override by its ID'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The override ID to delete
</ParamField>

## Response

Returns a success confirmation.

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/scope-overrides/permissions/override_456' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```

```javascript Node.js
const axios = require('axios');

const { data } = await axios.delete('https://api.example.com/scope-overrides/permissions/override_456', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
});
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true
}
```
</ResponseExample>
